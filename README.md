# ECC — Everything Claude Code

> **Nota**: Este fork funcionó como el espacio inicial para sentar las bases de la documentación de ECC en español. Ahora se mantiene como archivo histórico. Si buscas el repositorio original y su documentación actualizada, visita [affaan-m/ECC](https://github.com/affaan-m/ECC).

**ECC** is a **harness-native operator system for agentic work**, designed to be used within AI agent harnesses like **Claude Code**, **Cursor**, **OpenCode**, **Codex**, and others.

This repository was forked from the main [affaan-m/ECC](https://github.com/affaan-m/ECC) project to serve as a foundational workspace for **Spanish-language documentation**. It represented the first step in making the project's knowledge accessible to Spanish-speaking users, while respecting the MIT license of the original.

---

## About This Fork

The purpose of this fork is to provide a dedicated space for:
- Translating ECC's documentation, guides, and instructions into Spanish.
- Ensuring the quality and accuracy of the Spanish translations before contributing them back to the original repository.
- Collaborating with Spanish-speaking community members to improve accessibility.

For the original project, issues, and updates, please refer to the official repository: [github.com/affaan-m/ECC](https://github.com/affaan-m/ECC).

---

## About This Fork (Español)

Este fork del repositorio **ECC (Everything Claude Code)** nació como el primer paso para llevar la documentación al español. Fue el espacio inicial donde se sentaron las bases para que la comunidad hispanohablante pudiera acceder al proyecto en su propio idioma.

La iniciativa de este fork sirvió como motivación y punto de partida para que, posteriormente, otros colaboradores se sumaran y culminaran el esfuerzo de traducción. La idea siempre fue clara: que el conocimiento técnico no deba estar limitado por barreras idiomáticas.

Si bien el trabajo de traducción ya ha sido integrado en gran medida gracias al esfuerzo colectivo, este fork permanece como un testimonio del primer impulso que hizo posible la documentación en español. Para ver el estado actual del proyecto, visita el repositorio oficial: [github.com/affaan-m/ECC](https://github.com/affaan-m/ECC).

## What is ECC?

For the full description, installation, and usage of ECC, please visit the main repository.

In summary, ECC provides:
- **Agentic workflows** and operators.
- **Skills** for coding, design, testing, and security.
- **Hooks** and automations to streamline interactions.
- Cross-harness compatibility (Claude Code, Cursor, OpenCode, etc.).
