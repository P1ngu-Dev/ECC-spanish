# ECC — Everything Claude Code

> **Nota**: Este fork es un espacio para traducir y mejorar la documentación de ECC al español. Si buscas el repositorio original, visita [affaan-m/ECC](https://github.com/affaan-m/ECC).

**ECC** is a **harness-native operator system for agentic work**, designed to be used within AI agent harnesses like **Claude Code**, **Cursor**, **OpenCode**, **Codex**, and others.

This repository was forked from the main [affaan-m/ECC](https://github.com/affaan-m/ECC) project to serve as a dedicated workspace for **Spanish-language documentation**. It allows contributors to translate, validate, and improve the project's documentation for Spanish-speaking users, while respecting the MIT license of the original.

---

## About This Fork

The purpose of this fork is to provide a dedicated space for:
- Translating ECC's documentation, guides, and instructions into Spanish.
- Ensuring the quality and accuracy of the Spanish translations before contributing them back to the original repository.
- Collaborating with Spanish-speaking community members to improve accessibility.

For the original project, issues, and updates, please refer to the official repository: [github.com/affaan-m/ECC](https://github.com/affaan-m/ECC).

## About This Fork (Español)

Este fork del repositorio **ECC (Everything Claude Code)** fue creado para facilitar la traducción de la documentación al español.

La idea es que todo el mundo pueda usar esta documentación, y ya que el inglés no es el idioma nativo de muchas personas, hay personas que han hecho un esfuerzo extra para traducirla. Tú puedes ser uno de ellos si ves algo que faltan.

Aquí es donde nace este fork; gracias a la posibilidad de crear una copia del proyecto, se nos facilitó el uso de herramientas automáticas (por ejemplo, Model Context Protocol) para agilizar este proceso.

Si deseas ver el proyecto original y estar al día con actualizaciones, visita su repositorio oficial aquí: [github.com/affaan-m/ECC](https://github.com/affaan-m/ECC).

## What is ECC?

For the full description, installation, and usage of ECC, please visit the main repository.

In summary, ECC provides:
- **Agentic workflows** and operators.
- **Skills** for coding, design, testing, and security.
- **Hooks** and automations to streamline interactions.
- Cross-harness compatibility (Claude Code, Cursor, OpenCode, etc.).
