# Prompts de demostración de Hermes x ECC

## Prompt 1: ECC construye ECC

Usa el repositorio actual de ECC y el paquete público de lanzamiento en `docs/releases/2.0.0-rc.1/`.

Haz 4 cosas en orden:

1. Inspecciona el estado de git y el diff actual del repositorio, luego dame un resumen conciso de PR o release de ECC v2.0.0-rc.1 que pruebe que ECC se está usando para construir ECC mismo.
2. Finaliza un hilo sólido para X.
3. Finaliza una publicación sólida para LinkedIn.
4. Dime las 3 grabaciones exactas que debo hacer después, más lo que Hermes puede generar automáticamente después de que grabe.

Sé decisivo y práctico.

## Prompt 2: Convertir una grabación en assets

Asume que acabo de grabar:

- un gancho frente a cámara
- una captura de pantalla de Hermes usando ECC para publicar ECC v2.0.0-rc.1
- un recorrido de configuración del workspace de Hermes x ECC

Dame:

1. un plan de edición para formato corto para X, LinkedIn, TikTok y YouTube Shorts
2. un guion de voz en off por si quiero volver a grabar audio limpio
3. los nombres de archivo y carpetas exactos, relativos al repo, que debería usar para el material bruto
4. los assets que Hermes puede generar automáticamente después de que coloque los archivos en su lugar

Manténlo operativo.

## Prompt 3: Empuje de lanzamiento público

Usando el paquete de lanzamiento de ECC v2.0.0-rc.1, dame:

1. un tweet de lanzamiento
2. un tweet de seguimiento
3. un comentario corto para LinkedIn que pueda pegar debajo de la publicación
4. un breve handoff para Telegram que pueda enviar luego a Hermes para seguir distribuyendo este lanzamiento por otros canales

Haz que suene como un operador entregando trabajo real, no como un cliché de hilo de lanzamiento.
