# Notas de la versión de ECC v1.8.0

## Posicionamiento

ECC v1.8.0 posiciona el proyecto como un sistema de rendimiento del entorno de ejecución de agentes, no solo como un paquete de configuración.

## Mejoras clave

- Se estabilizaron los hooks y el comportamiento del ciclo de vida.
- Se amplió la superficie de evaluación y operaciones de bucles.
- Se actualizó NanoClaw para uso operativo.
- Se mejoró la paridad entre harnesses (Claude Code, Cursor, OpenCode, Codex).

## Enfoque de actualización

1. Valida los valores predeterminados del perfil de hooks en tu entorno.
2. Ejecuta `/harness-audit` para establecer una línea base de tu proyecto.
3. Usa `/quality-gate` y los flujos de evaluación actualizados para aplicar consistencia.
4. Revisa las notas de atribución y licencias de los ecosistemas referenciados: [reference-attribution.md](./reference-attribution.md).
