# Atribución de referencias y notas de licencias

ECC v1.8.0 hace referencia a investigación e inspiración de flujo de trabajo provenientes de:

- `plankton`
- `ralphinho`
- `infinite-agentic-loop`
- `continuous-claude`
- perfiles públicos: [zarazhangrui](https://github.com/zarazhangrui), [humanplane](https://github.com/humanplane)

## Política

1. No copiar código directamente de fuentes sin licencia o incompatibles.
2. Las implementaciones de ECC se reescriben para la arquitectura y el modelo de licencias de este repositorio.
3. El material referenciado se usa solo como ideas, patrones y marco conceptual, salvo que la licencia permita explícitamente su reutilización.
4. Cualquier reutilización directa futura requiere verificación explícita de la licencia y atribución de la fuente en el archivo y en las notas de la versión.
