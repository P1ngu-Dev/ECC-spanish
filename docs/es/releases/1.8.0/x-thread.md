# Borrador de hilo de X - ECC v1.8.0

1/ ECC v1.8.0 ya está disponible. Esta versión trata de una sola cosa: mejor rendimiento del entorno de ejecución de agentes.

2/ Lanzamos correcciones de fiabilidad de hooks, comandos de operaciones de bucles y flujos de evaluación más sólidos.

3/ NanoClaw v2 ahora admite enrutamiento de modelos, carga dinámica de skills, branching, búsqueda, compactación, exportación y métricas.

4/ Si tus agentes rinden por debajo de lo esperado, empieza con `/harness-audit` y endurece los quality gates.

5/ La paridad entre harnesses sigue siendo una prioridad: Claude Code, Cursor, OpenCode, Codex.
